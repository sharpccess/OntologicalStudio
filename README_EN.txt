Ontological Studio - Quick Guide

Welcome to Ontological Studio.

This system lets you model universes, entities, relationships, and scenarios to enrich prompts and solve cases with AI support.

Main points:
- Create a universe and select it to enable the remaining tabs.
- Design entities and relationships from the canvas or side forms.
- Use hydration to enrich an entity description.
- Create scenarios and run AI-powered solutions.
- Save entities or full models into the library for reuse.
- Export artifacts as TXT, Markdown, PDF, or Word.

Key sections:
- Universes: main work container.
- Canvas: visual design of entities and connections.
- Entities: detailed editing and base management.
- Relationships: current relationship view.
- Scenarios: prompt and solution generation.
- Library: reusable entities and full models.
- AI: provider and model configuration.

Suggestion:
Keep each universe focused on one concrete system to get more precise prompts.
