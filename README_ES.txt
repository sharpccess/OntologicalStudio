Ontological Studio - Guía rápida

Bienvenido a Ontological Studio.

Este sistema permite modelar universos, entidades, relaciones y escenarios para enriquecer prompts y resolver casos con ayuda de IA.

Puntos principales:
- Crea un universo y selecciónalo para activar el resto de pestañas.
- Diseña entidades y relaciones desde el canvas o formularios laterales.
- Usa la hidratación para enriquecer la descripción de una entidad.
- Crea escenarios y ejecuta soluciones con la IA configurada.
- Guarda entidades o modelos completos en la librería para reutilizarlos.
- Exporta artefactos en TXT, Markdown, PDF o Word.

Secciones clave:
- Universos: contenedor principal de trabajo.
- Canvas: diseño visual de entidades y conexiones.
- Entidades: edición detallada y gestión base.
- Relaciones: vista de relaciones actuales.
- Escenarios: generación de prompts y soluciones.
- Librería: reutilización de entidades y modelos.
- IA: configuración del proveedor y modelo.

Sugerencia:
Mantén cada universo enfocado en un sistema concreto para obtener prompts más precisos.
